# MOVIE MANIA
## FULLY FUNCTIONING MOVIE APP 

This is a Fully functional movie app built upon **REACT** ,**REDUX** and **MUI**



## Features

- User authentication

- Search movies

- Dark mode

- category wise movie sorting
- Add to  favourite and watchlist



## Tech Stack

**Client:** React,Material UI , Redux, Router-dom
